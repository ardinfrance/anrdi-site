# ANRDI Site

Dépôt de travail du site ANRDI.

Ce repository permet de proposer des corrections, des améliorations et des refactorings sur le projet.

## Contribution

Les contributions peuvent porter sur :

- PHP
- HTML
- CSS
- JavaScript
- structure du projet
- correction de bugs
- amélioration de l'interface
- amélioration du code

## Workflow

```bash
git checkout -b feature/ma-modif
git add .
git commit -m "Ma modification"
git push -u origin feature/ma-modif
```

## Note

Certaines parties du projet dépendent d'éléments non présents dans ce dépôt.
